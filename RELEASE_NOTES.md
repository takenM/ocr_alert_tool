# OCR Alert Tool v1.0.0

## 🚀 主な機能
- **リアルタイム数値監視**: 画面上の指定エリアの数値をOCRで読み取り、基準値を超えたらアラート（音＋ダイアログ）で通知します。
- **マルチモニター対応**: 複数のディスプレイにまたがる領域や、サブモニター上の領域もスムーズに選択・監視可能。
- **高精度OCR**: 文字サイズが小さい場合でも自動的に拡大補正を行い、正確に数値を読み取ります。
- **デバッグログ機能**: アプリ画面内でリアルタイムに認識結果（Rawデータ/数値変換後）を確認できます。

## 📦 実行方法 (配布版)
1. `OCRAlertTool.exe` をダウンロードして任意のフォルダに置きます。
2. ダブルクリックして起動します。
3. 「範囲選択」ボタンを押し、監視したい数値エリアをドラッグして囲みます。
4. 閾値（アラートを鳴らす数値）を入力し、「監視開始」を押します。

## ⚠️ 注意事項
- **Tesseract内蔵**: このEXEファイルにはOCRエンジンが含まれているため、別途インストールの必要はありません。
- **誤検知**: 背景が複雑な場合や、文字色が背景と近い場合は認識精度が落ちることがあります。なるべく「数字だけ」が黒背景や白背景ではっきり表示されている箇所を選んでください。
